# GIT COMMANDS

| command | effect |
| --- | --- |
| `git clone _url_` |kopieren von einem Online-Repository am Anfang. Wir nur am anfang gebraucht |
| `git add .` |alle änderungen für ein Comit vorschlagen |
| `git commit -m "_message_"` |   änderungen in das lokale Repo commiten / eintragen  |
| `git pull` | letzte von Online-Repo runterladen  |
| `git push`  | eigene lokale Repo-Version hochladen  |
