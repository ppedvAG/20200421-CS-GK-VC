# GIT COMMANDS

| command | effect |
| --- | --- |
| `git add .` | alle Änderungen für ein Commit vorschlagen |
| `git commit -m "_message_"` | Änderungen in das lokale Repository commiten / eintragen |
| `git pull` | letzte Version vom Online-Repository herunterladen |
| `git push`  | alle eigenen lokalen Repository-Versionen hochladen  |
| `git clone _url_` | kopieren von einem Online-Repository|

[git homepage](https://git-scm.com)
